# Bibs — Claude Code Instructions

## Project
Plain HTML/CSS/JS only. Single file: index.html. No frameworks, no libraries, no build tools.

## How we work
- One small piece at a time. Do not build everything at once.
- After each change, stop and wait for review before continuing.
- Explain what you did and why after every change.
- If a refactor is needed, flag it and wait for approval — don't do it silently.

## Code style
- Keep code simple and readable.
- Add a short comment on anything non-obvious.
- No minification, no clever one-liners.
